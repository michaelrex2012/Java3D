Model

Model Info

.j3dm File Format

.j3dm files are formated by the following:

(ln 1) A comment about the file (MUST BE NON-MODEL DATA)
(ln 2) Tri 1, Vertex 1 (x, y, z)
(ln 3) Tri 1, Vertex 2 (x, y, z)
(ln 4) Tri 1, Vertex 3 (x, y, z)
(ln 5) Tri 1, Color (Decimal RGB color)
(ln ...) [Continue ln 2-5 for each desired triangle.]
